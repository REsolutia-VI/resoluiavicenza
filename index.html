Connetti Google Calendar

Per visualizzare le date disponibili è necessario autorizzare l'accesso al tuo calendario Google.

⚙ Configurazione una-tantum

Prima di iniziare, inserisci il tuo Google OAuth Client ID.
Per ottenerlo:

  1. Vai su console.cloud.google.com
  2. Crea un progetto (o selezionane uno esistente)
  3. Menu → API e servizi → Credenziali
  4. Crea credenziali → ID client OAuth 2.0
  5. Tipo applicazione: Applicazione web
  6. Origini JavaScript autorizzate: inserisci l'URL di questa pagina
  7. Copia il Client ID e incollalo qui sotto
  8. Attiva l'API Google Calendar dalla Libreria API
Accedi con il tuo account Google per autorizzare la lettura del calendario e la creazione di eventi. L'accesso viene usato esclusivamente per questo strumento.
